Snake The Game - Project for "Podstawy Programowania 2" module

Introduction

Project created for the "Podstawy Programowania 2" module at the Kielce University of Technology. The project is a simple implementation of the Snake game. The game is written in C using the Allegro 5 library.

Gameplay

Snake is controlled by the arrow keys o WASD keys. The goal of the game is to eat as many apples as possible without hitting the walls or the snake's body.



Download

Procject can be downloaded from the GitHub repository: https://github.com/kyzelm/PRO-PP-snake
