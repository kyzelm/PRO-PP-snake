\chapter{Snake The Game -\/ Project for "{}\+Podstawy Programowania 2"{} module}
\hypertarget{index}{}\label{index}\index{Snake The Game -\/ Project for "{}Podstawy Programowania 2"{} module@{Snake The Game -\/ Project for ""Podstawy Programowania 2"" module}}
\hypertarget{index_intro_sec}{}\doxysection{\texorpdfstring{Introduction}{Introduction}}\label{index_intro_sec}
Project created for the "{}\+Podstawy Programowania 2"{} module at the Kielce University of Technology. The project is a simple implementation of the Snake game. The game is written in C using the Allegro 5 library.\hypertarget{index_gameplay_sec}{}\doxysection{\texorpdfstring{Gameplay}{Gameplay}}\label{index_gameplay_sec}
Snake is controlled by the arrow keys o WASD keys. The goal of the game is to eat as many apples as possible without hitting the walls or the snake\textquotesingle{}s body. The game has a simple menu with the option to start the game, see the highscore list, and exit the game.\hypertarget{index_creators_sec}{}\doxysection{\texorpdfstring{Creators}{Creators}}\label{index_creators_sec}
